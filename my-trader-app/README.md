# 📊 Trader App — Фронтенд (React)

Это клиентская часть приложения для изучения трейдинга. Позволяет просматривать обучающие материалы и проходить тесты.

## 🛠 Технологии

- **React** (Create React App)
- **CSS** (возможно с использованием CSS-модулей или сторонних библиотек)

## 🚀 Как запустить локально

1. Убедитесь, что у вас установлены [Node.js](https://nodejs.org/) и [Yarn](https://yarnpkg.com/).
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/majo1242/my-trader-project.git
   cd my-trader-app
